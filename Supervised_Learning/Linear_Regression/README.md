# Insurance Cost Prediction

This project uses Linear Regression to predict medical insurance charges.

## 📊 Dataset Features

* Age
* Sex
* BMI
* Children
* Smoker
* Region

## 🎯 Target

* Charges (insurance cost)

## ⚙️ Steps

* Data loading
* Data preprocessing (encoding categorical features)
* Train-test split
* Model training using Linear Regression
* Model evaluation using R² score

## 🛠️ Tools Used

* Python
* Pandas
* Scikit-learn
* Seaborn

## 📌 Result

The model predicts insurance charges based on user details.





# House Price Prediction

This project uses Linear Regression to predict house prices.

## 📊 Dataset Features

* LotArea (land size)
* OverallCond (condition)
* YearBuilt
* TotalBsmtSF (basement area)
* MSZoning, LotConfig, BldgType (encoded)

## 🎯 Target

* SalePrice (house price)

## ⚙️ Steps

* Data cleaning (removed Id, handled missing values)
* Encoding categorical features
* Train-test split
* Model training using Linear Regression
* Evaluation using R², MAE, RMSE, MAPE
* Feature scaling for improvement

## 🛠️ Tools Used

* Python
* Pandas
* NumPy
* Scikit-learn

## 📌 Result

The model predicts house prices based on property features.
